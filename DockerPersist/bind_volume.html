Bind Volumes

  • Maps a host file or directory into container file or directory
  • Basically two locations pointing to the same file
  • Skips UFS and host file override in any container, same as Volumes
  • Can't be used in the Dockerfile, must be at container run
  • Eg, ... run -v /home/sachet/docker:/path/container
  • Use the below command to point the container's given directory into host current directory
    1. docker container run -d --name nginx_host -p 80:80 -v $(pwd):/usr/share/nginx/html nginx
